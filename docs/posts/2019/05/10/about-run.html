跑步那些事儿

发布时间 : 2019-05-10 最后修改 : 2019-06-14
上一篇 : 2019年书单 下一篇 : 《关于》

“妈妈说过，要往前走，就得先忘记过去。我想，这就是跑的用意。” ------《阿甘正传》

其实每个人都不想止步不前，但又不想忘记过去，更不想去跑。 如果你不知道做什么，那就开始跑步吧。 当你迈腿开始跑步的时候，其实心里所想并没有那么多。

# 而且

I. 跑步可以帮助你消除7中病痛

治疗偏头痛 治疗耳背 癌能手 拯救皮肤 缓解焦虑 修复膝盖软骨 增强记忆力

II. 坚持长期跑步的益处

祛湿 缓解眼部疲劳 改善颈椎以及肩部不适 改善新陈代谢 缓解闭眼 加速脂肪燃烧，改善腰部线条 减少妇科疾病，调节月经 排毒 调节内分泌

那么就开始跑步吧，跑步会让你忘掉不快乐不开心的事情，会让你专注于只做一件事，那就是往前跑。
